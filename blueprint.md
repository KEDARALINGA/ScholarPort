# **App Name**: ScholarPort

## Core Features:

- Student Registration: Allows new students to create accounts, providing their personal, academic, and financial details, and uploading necessary documents.
- Secure Student Login: Enables students to securely log in using their credentials and access their personalized dashboard.
- Scholarship Catalog: Displays a comprehensive list of available scholarships with basic details, offering filtering options based on criteria like eligibility or deadline.
- Scholarship Application Form: Provides a user-friendly form for students to apply to selected scholarships, including academic marks, income proof, and category selection.
- AI-Powered SOP Assistant: A generative AI tool to assist students in drafting a compelling Statement of Purpose (SOP) by suggesting content, structuring ideas, or refining language based on their scholarship and personal inputs.
- Application Status Tracker: Allows students to view the real-time status of their scholarship applications (e.g., 'Pending', 'Approved', 'Rejected').
- Admin Scholarship Manager: Provides administrative users with tools to add new scholarships, edit existing scholarship details, and remove obsolete scholarship listings.

## Style Guidelines:

- Color scheme: A light theme to convey clarity and professionalism, suitable for an educational portal.
- Primary color: Deep indigo (#5533CC), chosen for its association with wisdom, integrity, and depth in an educational context, ensuring strong contrast against a light background.
- Background color: A subtle hint of primary hue with high desaturation (#F8F8FC) for a clean, unobtrusive canvas.
- Accent color: A vibrant sky blue (#4CB2FF), an analogous color to the primary, to highlight key interactive elements and provide visual interest.
- Headline and Body font: 'Inter', a modern sans-serif typeface, selected for its high legibility across devices and professional, objective appearance.
- Utilize clean, modern line-art icons that complement the educational theme and enhance readability without visual clutter.
- Responsive, grid-based layout prioritizing content hierarchy and readability. Use cards for scholarship listings and application summaries, and tables for data presentation (e.g., admin panels, application statuses).
- Subtle, functional animations for feedback on user actions (e.g., form submissions, navigation transitions) to enhance user experience without being distracting.
